# 🛡️ SafeSpeak AI

🚀 Stay Safe • Speak Smart • Understand the Internet

## 📌 Overview
SafeSpeak AI is a simple but powerful tool designed to protect users from digital scams and help them understand modern online language.

## 🔍 Key Features

- Scam detection using keyword-based analysis  
- Risk scoring system (visual + percentage)  
- Explanation of why a message is dangerous  
- “Grandma Mode” to translate Gen Z slang & emojis  
- Multi-language support (English, Bangla, Hindi)  

## 🌍 Real-World Impact

Digital scams are increasing rapidly, especially in developing regions.  
SafeSpeak AI helps users identify threats and understand online communication more clearly.

## 🧠 How It Works

- Detects suspicious keywords (OTP, bank, urgent, etc.)
- Assigns a risk score
- Explains threats in simple language
- Translates slang and emojis into understandable meanings

## 🛠️ Built With

- Python  
- Streamlit  

## 🎯 Purpose

This project focuses on awareness, prevention, and accessibility, making digital safety tools easier to use for everyday people.
